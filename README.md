Sistema de Levantamientos de Maquinaria Pesada

Actualmente, en el área de mantenimiento de maquinaria pesada, las inspecciones se
realizan de manera manual o poco estructurada, lo que genera diversos problemas:
Falta de estandarización en los levantamientos técnicos.
Pérdida o mala interpretación de información.
Alto margen de error humano.
Dificultad para priorizar fallas críticas.
Falta de seguimiento adecuado.

El proyecto busca resolver estos problemas mediante una herramienta digital que per-
mita realizar inspecciones más visuales, organizadas y confiables.

Objetivo del sistema

Desarrollar un sistema que permita registrar inspecciones de maquinaria pesada de
forma estructurada, clasificando fallas según su criticidad y generando reportes claros
para apoyar la toma de decisiones.
Justificación
Este sistema es necesario debido a la falta de control real sobre las condiciones de la

maquinaria pesada, lo cual puede derivar en fallas críticas, accidentes o pérdidas econó-
micas. El uso de una herramienta digital permitirá mejorar la eficiencia, reducir errores y

optimizar el mantenimiento.

Además, al tratarse de un tema específico, es importante que el proyecto documente
claramente el contexto del problema, el flujo de trabajo y la utilidad de una inspección
más gráfica y estructurada, de manera que cualquier desarrollador pueda comprenderlo y
llevarlo a cabo.
